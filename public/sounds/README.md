# Background Music (BGM)

## Required File
- **File**: `bgm-ambient.mp3`
- **Size**: < 500KB
- **Format**: MP3
- **Loop**: Must be seamless loop (1-3 minutes)
- **Style**: Soft, ambient, uplifting, family-friendly

## Recommended Sources (Royalty-Free)

### Option 1: Incompetech (CC-BY)
https://incompetech.com/music/royalty-free/music.html
- "Wallpaper" by Kevin MacLeod (light, playful)
- "Carefree" by Kevin MacLeod (upbeat, positive)

### Option 2: Pixabay Audio (CC0)
https://pixabay.com/music/
- Search: "calm ambient loop"
- Filter: No attribution required

### Option 3: YouTube Audio Library
https://www.youtube.com/audiolibrary
- Filter: Genre = Ambient

## How to Add
1. Download MP3 file from one of the sources above
2. Optimize with Audacity (compress to < 500KB if needed)
3. Rename to `bgm-ambient.mp3`
4. Place in this directory (`public/sounds/`)

## Note
The BGMPlayer component is already configured to load this file.
It will fail gracefully if the file is missing (no audio, but no errors).
